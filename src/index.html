A little bit about me..

I have always found joy in making and creating. For as long as I can remember, during my free time you could find me baking, crafting, playing the violin or just using my hands in some way. The sense of achievement you feel after seeing the finished product of something you have made never ceases to amaze me. When I learned about web development, and started dabbling around with my own projects, I was instantly hooked. It felt like I had found my purpose. I am overjoyed to have created a life where I get to build amazing things for others to use and enjoy.

I am currently working as a front-end developer for a small startup that focuses on email marketing. I have experience working in HTML, CSS and Javascript. I am currently learning React. Github is my preferred version control system, but I am comfortable using Bitbucket. The terminal integration in VSCode makes my heart sing. Some of my recent projects include connecting a map api to a weather data api to show the current weather for any location clicked on. And building out my own wedding website, complete with the ability to filter suggested locations to visit by various parameters as well as an RSVP section. Check out my projects and learn a bit more about each one by visiting my portfolio page.
